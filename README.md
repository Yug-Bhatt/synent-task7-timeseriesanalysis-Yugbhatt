# Time Series Analysis using Apple Stock Price Dataset

## Internship Project

This project performs Time Series Analysis on Apple stock price data using Python.

## Features
- Trend Analysis
- Rolling Mean & Rolling Standard Deviation
- Seasonality Detection
- Time Series Decomposition
- Correlation Heatmap
- ARIMA Forecasting
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- Scikit-learn

## Dataset
Apple Stock Price Dataset
